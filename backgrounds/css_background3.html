Caixa transparente

Quando usar a propriedade de opacidade para adicionar transparência para o fundo de um elemento, todos os seus elementos filhos também se tornam transparentes. Isto pode tornar o texto contido num elemento transparente difícil de ler:

opacidade 0.1

opacidade 0.3

opacidade 0.6

opacidade 1 (padrão)
